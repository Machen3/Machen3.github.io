Vim模式切换和基本命令

Vim模式

通过Vim命令进入Vim后，直接就是Normal model。

Normal model
正常模式下无法编辑,可通过按ESC2次返回

Insert model
插入模式下可以修改内容,可通过I\i\a\A等键可以启动

Command model
命令模式下无法编辑内容，但可以使用其他的相关命令：

  • 比如dd：删除单行

Normal model
模式下输入(英文冒号):开启 模式。可以试用相关命令：

  • 退出命令q
  • 保存命令w
  • 保存并推出命令wq
  • 强制执行！

Visual model
可视模式，在Command model下按V就可以实现。
